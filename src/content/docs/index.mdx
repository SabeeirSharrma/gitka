## Why Gitka?

<CardGrid>
	<Card title="Offline Commits" icon="document">
		Extract a repo temporarily, let you commit offline, and recompress when you're done. Perfect for working completely off-the-grid.
	</Card>
	<Card title="Aggressive Compression" icon="rocket">
		Automatically selects zstd tiers based on available space and deduplicates common blobs across multiple repositories.
	</Card>
	<Card title="LAN Sharing" icon="laptop">
		Serve a repository directly to others on your local network via GitFlare without needing internet access.
	</Card>
	<Card title="Resilient & Secure" icon="puzzle">
		Optional par2 recovery records for corruption protection, AES-256-GCM encryption, and crash resilience with auto-recompression.
	</Card>
	<Card title="Cross-Platform" icon="laptop">
		Works on Linux, macOS, and Windows with unified installers. Auto-detects default branches and merges cleanly.
	</Card>
</CardGrid>
