# Installation

Gitka builds from source using Rust. We provide install scripts for all platforms — they handle dependencies automatically.

## Quick Install

### Linux / macOS

```bash
curl -sSf https://sabeeir.qd.je/gitka/install.sh | bash
```

### Windows (PowerShell)

Run as Administrator or ensure your user profile is in PATH:

```powershell
[Net.ServicePointManager]::SecurityProtocol = 'tls12'
iex "& { $(irm https://sabeeir.qd.je/gitka/install-windows.ps1) }"
```

Or download and run locally:

```powershell
powershell -ExecutionPolicy Bypass -File install-windows.ps1
```

The Windows installer will automatically install:
- **Git for Windows** (if not already installed)
- **Rust/Cargo** via rustup (if not already installed)
- **Visual Studio 2022 Build Tools** with C++ workload (if not already installed)

### Custom Install Location

**Linux/macOS** — use the `--prefix` flag:

```bash
curl -sSf https://sabeeir.qd.je/gitka/install.sh | bash -s -- --prefix ~/.local
```

**Windows** — pass the `-InstallDir` parameter:

```powershell
.\install-windows.ps1 -InstallDir "D:\Tools\Gitka"
```

## Requirements

- **Rust toolchain** (cargo) — the installer handles this if missing
- **Git** — needed for cloning repositories
- **C compiler** — GCC/Clang on Linux/macOS, MSVC on Windows (installer handles MSVC)
